# Spotify Playlist Cleaner

## Work in progress. This is just a hobby project that is not working yet.

This application is designed to help keep your Spotify playlists fresh and up-to-date. It automatically removes any track that is older than one month, ensuring that your playlists are always filled with the latest songs.

## Features
* Automatic removal of tracks older than one month
* Seamless integration with your Spotify account
* Easy to use interface

## How to Use
* Connect your Spotify account
* Select the playlists you want to keep clean

The app will automatically remove tracks older than one month. Enjoy your always fresh Spotify playlists!
